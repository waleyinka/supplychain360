# SupplyChain360 Data Platform

## Objective

Design and implement a production-grade data platform that:

 - Centralizes fragmented supply chain data
 - Enables real-time and historical analytics
 - Improves inventory and logistics efficiency

## Tech Stack
 - Orchestration: Airflow
 - Storage: AWS S3 (Parquet)
 - Warehouse: Snowflake
 - Transformation: dbt
 - IaC: Terraform
 - Containerization: Docker

## Architecture Principles
 - Idempotent pipelines
 - Schema evolution handling
 - Incremental processing
 - Separation of concerns
 - Observability first


## Key Business Outputs
 - Stockout trends
 - Supplier performance metrics
 - Warehouse efficiency KPIs
 - Regional demand forecasting
